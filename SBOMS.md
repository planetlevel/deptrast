# SBOM Gallery

**Explore real-world SBOMs from popular Docker images**

This gallery contains 170+ CycloneDX SBOMs from the [Anchore SBOM Examples](https://github.com/anchore/sbom-examples) repository. The links below will instantly load and explore the dependency tree in the Deptrast SBOM Explorer.

**Want to explore your own SBOM?** Just drag and drop any CycloneDX JSON file onto the viewer, or paste a URL!

**Want to make your own SBOMs?** Just visit the [Deptrast GitHub Page](https://github.com/planetlevel/deptrast) and download the CLI!
